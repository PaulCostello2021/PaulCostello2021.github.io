HTML Forms Reference

Form Tags

Tag Name Description
<form> Form

The HTML <form> element is used to collect and send information to an external source.
It can contain various input elements When a user submits the form, information in these
input elements is passed to the source which is named in the action attribute of the form.

<input> Input

The HTML <input> element is used to collect input from the user in different formats
such as radio button, number, checkbox etc

<txtarea> Txtarea

The HTML <txtarea> element is used when creating a text-box for multiple line input.
The element supports the rows and cols attributes which determine the height and width,
respectively of the element.
<select> Select

The HTML <select> element can be used to create a dropdown list.
<datalist> datalist

The HTML <datalist> element is used to store a list of options.

Form Attributes

Attribute Name Description
type type

The HTML type attribute allows the user to type censored text inside the field. It is used to
type in sensitive information.
required required

The HTML required attribute specifies that the field must include a value.
max max

The HTML max attribute specifies specifies the maximum value for the input field.
